\# AquaMonitor - Sistem za pametno upravljanje resursima



\## Svrha projekta

AquaMonitor je open-source IoT rješenje dizajnirano za praćenje kvalitete vode i potrošnje u realnom vremenu. Projekt omogućava korisnicima da putem senzora prikupljaju podatke, čuvaju ih u bazi podataka i vizuelizuju trendove kroz intuitivan web interfejs.



\## Tehnologije

\* Backend: Node.js (Express) / Python (FastAPI)

\* Frontend: React.js (Tailwind CSS)

\* Database: MariaDB / InfluxDB (za vremenske serije)

\* Infrastructure: Docker kontejneri i Nginx

\* Komunikacija: MQTT protokol i REST API



\## Opis

Sistem podržava slanje automatskih obavještenja ukoliko parametri pređu definisane kritične vrijednosti. Arhitektura je modularna, što omogućava lako dodavanje novih tipova senzora i proširenje dashboarda.
